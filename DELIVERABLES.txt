HOW TO USE
================================================================================

1. START BACKEND
   cd backend
   python main.py

2. TEST ENDPOINT
   - Browser: http://localhost:8000/api/v1/dashboard
   - Swagger: http://localhost:8000/docs
   - cURL: curl http://localhost:8000/api/v1/dashboard

3. INTEGRATE IN FRONTEND
   fetch('http://localhost:8000/api/v1/dashboard')
     .then(r => r.json())
     .then(data => {
       // Use data.stats, data.courses, etc
     })

See FRONTEND_INTEGRATION_GUIDE.md for complete React examples.
