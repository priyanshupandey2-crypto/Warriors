   • src/styles/variables.css     → All color tokens, typography, spacing

🧩 Components Ready to Use
   UI Primitives: Button, Input, Textarea, Select, Card, Badge, Loader
   Domain: StatsCard, CourseCard
   Layout: Header, Sidebar, AppShell

🏪 State Management (Zustand)
   • authStore      → User, auth token, login/logout
   • courseStore    → Courses, enrollment, drafts
   • generationStore → AI generation progress
   • uiStore        → Sidebar, theme, notifications

🪝 Hooks for Components
   • useAuth()      → Login, signup, user state
   • useCourses()   → Fetch courses, create, enroll
   • useUI()        → Sidebar toggle, notifications

📝 Types (Fully Typed)
   • Course, User, LoginRequest, etc.
   • Ready for backend integration

🔧 API Setup
   • Axios client with auth interceptors
   • Ready for `NEXT_PUBLIC_API_URL` env var

USAGE EXAMPLES
═══════════════════════════════════════════════════════════════

Import Components:
   import { Button, Input, Card } from '@/components/ui'
   import { Header, Sidebar, AppShell } from '@/components/layout'
   import { CourseCard } from '@/components/domain'

Use State:
   const { user, login } = useAuthStore()
   const { courses, enrollCourse } = useCourseStore()

Call Hooks:
   const { user, isAuthenticated } = useAuth()
   const { showNotification } = useUI()

Build Pages:
   import { AppShell, Header, Sidebar } from '@/components'
   import { Button, Input } from '@/components/ui'
   import { useAuth } from '@/hooks'

   export default function MyPage() {
     const { user } = useAuth()
     return (
       <AppShell>
         <h1>Hello, {user?.name}</h1>
       </AppShell>
     )
   }

GETTING STARTED
═══════════════════════════════════════════════════════════════

1. Install dependencies:
   $ npm install

2. Start development server:
   $ npm run dev

3. Open http://localhost:3000

4. Start building pages using the foundation!
